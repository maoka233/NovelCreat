# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-17

### Added
- Initial release of NovelCreat
- Electron + React + TypeScript desktop application
- DeepSeek AI integration for novel writing assistance
- Chapter management system
- Outline generator with style selection
- Chapter editor with AI-powered content generation
- Knowledge base system for characters, world settings, and plot tracking
- Context management for maintaining narrative consistency
- Storage service with SQLite/JSON support
- Dark theme UI with Tailwind CSS
- Basic project documentation and configuration files

### Features
- **AI-Powered Writing**: Integration with DeepSeek API for intelligent content generation
- **Chapter Management**: Create, edit, and organize novel chapters
- **Outline Generation**: Automatically generate story outlines based on descriptions
- **Context Awareness**: Maintains story context for consistent narrative generation
- **Knowledge Base**: Track characters, world settings, plot points, and consistency rules
- **Modern UI**: Clean, dark-themed interface with smooth animations

### Technical
- Electron 31.x for cross-platform desktop support
- React 18 with TypeScript for type-safe UI development
- Zustand for state management
- Vite for fast development and building
- Tailwind CSS for styling
- ESLint and Prettier for code quality
- electron-store for persistent data storage
